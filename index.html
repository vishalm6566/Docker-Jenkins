Jenkins Installation and Build Guide

Installation

To install Jenkins, follow these steps:


# Download the key
sudo wget -O /usr/share/keyrings/jenkins-keyring.asc https://pkg.jenkins.io/debian-stable/jenkins.io-2023.key

# Add the key into apt repository
echo "deb [signed-by=/usr/share/keyrings/jenkins-keyring.asc]" \
    https://pkg.jenkins.io/debian-stable binary/ | sudo tee \
    /etc/apt/sources.list.d/jenkins.list > /dev/null

# Install Jenkins
sudo apt-get update
sudo apt-get install fontconfig openjdk-17-jre
sudo apt-get install jenkins

# Check the Jenkins service status
sudo systemctl status jenkins

# Start the Jenkins service
sudo systemctl start jenkins

# Enable Jenkins service to start on boot
sudo systemctl enable jenkins

# Visit the localhost:8080 URL to install required Jenkins plugins

Build Using Jenkins

To build using Jenkins, use the following commands:


# Remove existing service
docker service rm mywebsite

# Remove existing image
docker image rm mywebsite

# Build the image with the latest changes
docker build -t mywebsite .

# Create a service for running the latest changes
docker service create --name mywebsite --replicas 5 -p 8000:80 mywebsite
